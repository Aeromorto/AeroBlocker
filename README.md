# AeroBlocker
Bloqueador de anúncios e rastreadores. Privado, seguro, confiável.


## Como instalar

1. Clica no botão verde **Code** acima
2. Clica em **Download ZIP**
3. Extrai a pasta ZIP no seu computador
4. Abre o Edge e acessa: edge://extensions
5. Ativa o **Modo do desenvolvedor** (canto inferior esquerdo)
6. Clica em **Carregar sem pacote**
7. Seleciona a pasta extraída
8. Pronto! O ícone do AeroBlocker aparece na barra do Edge
